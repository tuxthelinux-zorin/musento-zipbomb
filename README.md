## WARNING: ZIP BOMB RESEARCH

This repository discusses or references zip bombs.  
**No actual harmful archive is included in this repository.**

If you plan to work with real zip bomb samples, do so only in a fully isolated environment such as:
- A disposable virtual machine
- An air-gapped test system
- Hardware you do not mind damaging or wiping

Do NOT extract or test any suspicious or unknown archives on a personal or production system.  
Proceed only if you understand the risks and take appropriate precautions.
